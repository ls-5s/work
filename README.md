关于 `goods 和Vue3` 项目的安装步骤，可参考项目的 `README.md` 信息，具体如下：

1. **前置条件**  
   确保已安装 `pnpm` 包管理工具（项目依赖管理基于 `pnpm`）。

2. **获取项目代码**  
   先将项目代码克隆或下载到本地，并进入项目根目录（即包含 `package.json` 和 `pnpm-lock.yaml` 的目录）。

3. **安装依赖**  
   在项目根目录下，执行以下命令安装项目所需的依赖：
   ```sh
   pnpm install
   ```

执行上述命令后，`pnpm` 会根据 `pnpm-lock.yaml` 中的依赖信息，自动下载并安装项目所需的所有依赖包，完成项目的安装配置。

# goods简介（核心解释在对应的readmd.md中）
题目一：基于商品白底图的智能主图/详情图生成系统

背景：
作为AI算法工程师，你需要开发一个智能体，能够根据商品的白底图和文本介绍，自动生成符合电商平台要求的高质量主图和详情图。

输入：
1.商品白底图（纯净背景，商品主体清晰）
2.商品文本介绍（标题、卖点、规格参数等）

输出：
1.3张不同风格的主图（800×800px）
2.5张详情图组合（宽度750px，高度自适应）
